Wat is javascript?

Javascript is een script die op je computer loopt, daarom wordt het client side language genoemd. Javascipt moet men niet gebruiken
als een "enhancer voor audiance experience". Daar bedoel ik mee dat men niet op Javascript moet leunen als een functionaliteit op de website.
Dit komt omdat de "user" de de functie "Javascript" moet inschakelen om het te kunnen te kunnen gebruiken.
Dit leidt tot fouten in sommige webbrowsers, daarom is het niet slim om Javascript te gebruiken om de functionaliteit van je website totaal te beheersen.

Belangrijk om te weten over Javascript!

Javascript loopt zijn script af van top tot bottom.

charAt()

Voorbeeld: let anyString = 'Brave new world'; console.log("The character at index 0 is '" + anyString.charAt() + "'"); // No index was provided, used 0 as default console.log("The character at index 0 is '" + anyString.charAt(0) + "'"); console.log("The character at index 1 is '" + anyString.charAt(1) + "'"); console.log("The character at index 2 is '" + anyString.charAt(2) + "'"); console.log("The character at index 3 is '" + anyString.charAt(3) + "'"); console.log("The character at index 4 is '" + anyString.charAt(4) + "'"); console.log("The character at index 999 is '" + anyString.charAt(999) + "'"); De methode charAt () returns het teken op de opgegeven index in een tekenreeks. Deze methode gebruikt de default value 0 als beginwaarde en elk caharacter in de zin heeft zijn eigen indexwaarde.
Als je dus opzoek bent naar de character in index 2 krijg je de letter "a".

concat()

In de cursus die ik volg betekent contactenate oftewel concat - het joinen van twee waardes. const str1 = 'Hello'; const str2 = 'World'; console.log(str1.concat(' ', str2)); // expected output: "Hello World" console.log(str2.concat(', ', str1)); // expected output: "World, Hello"

slice()

let email = "issakarissefabiala@gmail.com"; let result = email.slice(0,5); console.log(result); Eerste argument 0 is waar we van snijden en het tweede argument is waar we naar toe 5 snijden.

split()

const str = 'The quick brown fox jumps over the lazy dog.'; const words = str.split(' '); console.log(words[3]); // expected output: "fox" const chars = str.split(''); console.log(chars[8]); // expected output: "k" Het split niet iets in twee of meerdere delen, het pakt 1 deel split het van de andere delen. Dat zie je in de expected outputs.

subst()

Voorbeeld: let email = 'mario@thenetninja.co.uk'; let result = email.substr(4,10); Dit is iets anders dan slice, omdat het begint op de positie die in het eerste argument wordt vermeld, maar het telt verder tot 10 tekens. Dus hij neemt de waarde van positie 4 tot 14, omdat het vanaf 4 tot 14 telt in totaal.
