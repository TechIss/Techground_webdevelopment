Hoe geef je 2 of meer parameters mee aan een functie? Geef een voorbeeld. Wat zijn rest parameters? Geef een voorbeeld. Wat is het verschil tussen rest parameters en expliciete parameters die je aangeeft bij een functie?

Antwoord:

Function parameters are the names listed in the function definition. Function arguments are the real values passed to (and received by) the function. function functionName(parameter1, parameter2, parameter3) { // code to be executed }
