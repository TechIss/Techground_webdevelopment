Maak nu een Person object door gebruik te maken van Object initializers. Wat zou eventueel de voor en nadelen t.o.v Object constructors?

var obj = { name: "value", .... } De voorbeeld hierboven heb ik ergens gevonden. Het lijkt mij dat initializers betekent: Een object initializer is een uitdrukking waarmee we een nieuw gemaakt object kunnen initialiseren. Het is een door komma's gescheiden lijst van nul of meer paren van eigenschapsnamen en bijbehorende waarden van een object tussen een paar accolades {}.
