  1. Table
    2. Het containerelement voor tabelmodellen / tabelgegevens. Staat voor gegevens met meer dan één dimensie in de vorm van een tabel.
  2. Thead
    3. Wordt gebruikt om koptekstinhoud in een HTML-tabel te groeperen.
  3. tbody
    4. Wordt gebruikt om de body-inhoud in een HTML-tabel te groeperen.
  4. tfoot
    5. wordt gebruikt om voettekstinhoud in een HTML-tabel te groeperen.
  5. th
    6. defines a header cell in an HTML table.
  6. tr
    7. tag definieert een rij in een HTML-tabel.
  7. td
    8. tag defines a standard data cell in an HTML table. An HTML table has two kinds of cells: Header cells - contains header information (created with the th - element) Data cells - contains data (created with the td-tag element) The text in td-tag elements are regular and left-aligned by default. The text in th-tag elements are bold and centered by default.
