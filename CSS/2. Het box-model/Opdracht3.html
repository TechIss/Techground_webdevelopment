Opdracht 3

Wat gebeurt er als je een width en een height toevoegt aan de span tag ( <span> ) van vorige opdracht?

Dit is een block box en neemt de hele breedte in waardoor de volgende box op de volgende regel zit.

In dit block zit inline tekst in wat dus niet op de volgende regel komt te te staan.

Omdat het een inline tekst is, beinvloedt de span tag de regel niet. De box heeft geen outer-display waardoor het niks beinvloedt.
