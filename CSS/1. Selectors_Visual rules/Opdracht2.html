CSS pas je toe met een bepaalde syntax. Hoe ziet zo'n syntax eruit? Kun je ook uitleggen wat elk element uit de syntax betekent?

Syntax:
Syntaxis (ook wel grammatica genoemd) is een term uit de informatica. Voor een programmeertaal is de syntaxis de "taalregels" van het programmeren. Onder taalregels verstaan we de codes die een programmeertaal hanteert.

De Selector is het punt dat je in je HTML wilt stylen.
De decloration block is waar 1 of meer decleraties worden gesepareerd doormiddel van semicolons.
Elke decloration heeft een CSS property naam (color/width/height/background-color/font-family etc.) en een value (wat het property uiteindelijk moet zijn) en deze moeten gesepareerd worden met een colon. ";"
Bij het gebruik van een CSS syntax met meerdere declerations, moet men gebruik maken van het semicolons teken om de value te sluiten, en elke decleration blocks begint met een opening tag en sluiting tag.
In CSS gebruikt men de curly brace. De opening tag is: "{" & de sluiting tag is: "}".
