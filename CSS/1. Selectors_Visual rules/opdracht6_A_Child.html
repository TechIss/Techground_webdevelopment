Child Selector


Wat is een child selector & hoe gebruikt men dat?

Een child selector: ">" staat tussen twee CSS Selectors. Voorbeerd:
ul > li { margin: 0 0 5px 0; } Het zorg er voor dat de eerste lvl van de selector wordt geselecteerd en hiermee kan je in CSS de eerste lvl veranderen zonder de diepste lagen te veranderen.

  • item 1 ik gebruik: ul > li { background-color: rgb(38, 206, 32); }
    • Item A ik gebruik: ul > li { background-color: rgb(38, 206, 32); }
      1. Item A ik gebruik ul li { background-color: aqua; }
      2. Item B ik gebruik ul li { background-color: aqua; }
  • Item 2 ik gebruik: ul > li { background-color: rgb(38, 206, 32); }
  • Item 3 ik gebruik: ul > li { background-color: rgb(38, 206, 32); }
    • Item Aik gebruik: ul > li { background-color: rgb(38, 206, 32); }
      1. Item A ik gebruik ul li { background-color: aqua; }
      2. Item B ik gebruik ul li { background-color: aqua; }
    • Item B ik gebruik: ul > li { background-color: rgb(38, 206, 32); }
