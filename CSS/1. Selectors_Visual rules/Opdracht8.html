Cascading In CSS


Cascading in het Nederlands vertaald betekent trapgewijs. Als we deze betekenis aanhouden is de betekenis in HTML een volgorde aan regels dat CSS aanhoudt wanneer het
een bepaalde formule aan moet houden. Het kan zou zijn dat de gecreerde formule niet naar verwachting voldoet. Het is een "must" om te weten hoe de mechanisme werkt.


Inheritance


Inheritance in het Nederlands vertaald betekend erfenis. Inheritance is het erfen van values in het element's parent element. Dit kan verzorgen dat de code niet werkt naar verwachting


Voorbeeld 1: Stylesheet cascading

Bij het invullen van een code in CSS zijn er bepaalde regels waar men rekening moet houden

  1. Twee codes met dezelfde waarde
  2. Een element selector selecteert specifiek alle selectors met dezelfde waarde
  3. Een class selector (div clas="") selecteert een specifieke waarde

Twee codes met dezelfde waarde in css, zal niet allebei geselecteerd worden. De onderste formule wordt hierbij geselecteerd volgens de mechanisme
Bij een element selector en een class selector zal de browser luisteren naar de specifieke selector, omdat deze wat krachtiger is volgens regels
