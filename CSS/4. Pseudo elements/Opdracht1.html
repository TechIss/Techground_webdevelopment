Opdracht 1

Kun je enkele voorbeelden geven van pseudo-elementen (minimaal 4)? Beschrijf met je eigen woorden wat deze pseudo-elementen inhouden.

Antwoord: Is een pseudoelement omdat het wel iets target, maar iets nieuws creëert.

::before Creëert een nieuwe element achter de content. ::after Creëert een nieuwe element achter de vorige content. ::placeholder Targets de input type: "placeholder" ::grammar-error Targets de gramatica en onderlijnt het wanneer het woord verkeerd is geschreven.
