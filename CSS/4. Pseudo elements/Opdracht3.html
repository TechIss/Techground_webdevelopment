::first-line

Het :: first-line CSS pseudo-element past stijlen toe op de eerste regel van een block-level element. (br)
Merk op dat de lengte van de eerste regel van veel factoren afhangt, waaronder de breedte van het element, de breedte van het document en de lettergrootte van de tekst.

::file-selector-button

Het :: file-selector-button CSS pseudo-element vertegenwoordigt de knop van een <input> van type = "file". Het zorgt ervoor dat de knop werkt.
